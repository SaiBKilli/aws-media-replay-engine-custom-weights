# SegmentFromTranscription #

**MRE Plugin Class**
- Segmenter

**Description**:

This plugin creates segments based on spoken phrases identified from transcribed text.

**Applies to Media Type**:
- Video

**Use Cases**:
- To perform very basic functional tests of the MRE control plane.
